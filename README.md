## Personal Responsive Resume Website
A professional, single-page resume built with semantic HTML5 and CSS3 Flexbox.

## Tech Stack
* **HTML5**: Structured with semantic elements for SEO and accessibility.
* **CSS3**: Styled using Flexbox for a modern, split-column layout.
* **Google Fonts**: Integrated 'Montserrat' and 'Open Sans' for enhanced typography.

## Key Features
* **Interdisciplinary Layout**: A 2-column design that separates personal details from professional history.
* **Visual Skill Bars**: Dynamic CSS-based progress bars to represent technical proficiency.
* **Responsive Design**: Clean and readable across different screen resolutions.
* **Startup-Focused Branding**: A career objective tailored for high-growth, fast-paced environments.

## Project Structure
* `index.html` - The core structure and content.
* `style.css` - All layout, color, and typography rules.
* `Image.jpg` - Profile headshot.
